# AI Anki

Интерактивные карточки с проверкой ответов через Ollama: свободная формулировка, подсказки и уточняющие вопросы.

## Запуск

```bash
cd ~/courses/python/ai-anki
./run.sh
```

При старте открывается **главный экран** с прогрессом по каждой колоде. Выберите колоду и режим:

| Режим | Описание |
|-------|----------|
| **Повторить** | Новые карточки + те, у которых наступил срок повторения |
| **Только новые** | Ещё не изученные |
| **Слабые места** | Карточки с лучшим баллом ниже порога |
| **Вся колода** | Все карточки подряд |

## Прогресс и история

Данные хранятся в `~/.ai-anki/`:

| Файл | Содержимое |
|------|------------|
| `progress.json` | Баллы, интервалы повторения, даты, история ответов (до 50 на карточку) |
| `decks.json` | Список зарегистрированных колод |

**Интервальное повторение:** при хорошем ответе интервал растёт (1 д → 2.5 д → …); при ошибке карточка возвращается через несколько минут. Кнопка **«История карточки»** в сессии показывает все прошлые ответы.

## Настройка Ollama

1. **URL** — `http://ollama.webmastermsk.ru:30068` (прямой Ollama API).

2. **API-ключ** — не нужен для прямого Ollama; оставьте пустым. Для Open WebUI укажите ключ из *Settings → Account*.

3. **Модель** — `qwen3-coder:30b` (на сервере именно такое имя).

4. Нажмите **«Проверить Ollama»** — должны появиться доступные модели.

## Лимит уточнений

В настройках **«Макс. уточнений»** (по умолчанию 2) — сколько раз модель может задать дополнительный вопрос на одну карточку.

## Колоды

- Теория: Anki TXT или JSON с полями `question` / `reference`
- **Код**: JSON с `card_type: "code"`, полями `code`, `task`, `reference`
- При загрузке `~/Documents/anki-angular.txt` автоматически добавляются карточки из `decks/angular-code.json`
- **PHP 8+** (875 карточек): `decks/php8-plus.json`
- **Python — основы** (648 карточек): `decks/python-basics.json`
- **tkinter.ttk** (488 карточек, 50 с кодом): `decks/ttk-widgets.json`
- **PyTorch** (513 карточек, 65 с кодом): `decks/pytorch-basics.json`
- Перегенерация: `generate_python_deck.py`, `generate_ttk_deck.py`, `generate_pytorch_deck.py`

## Формат карточки с кодом

```json
{
  "card_type": "code",
  "question": "Краткое описание проблемы",
  "task": "Что нужно исправить или добавить",
  "code": "@Component({...})\nexport class X {}",
  "reference": "Правильный вариант кода"
}
```

1. Прочитайте вопрос и напишите ответ своими словами.
2. **Ctrl+Enter** или **«Проверить ответ»** — модель оценит ответ (0–100), даст обратную связь.
3. При ошибках — **подсказка**; при пробелах — **уточняющий вопрос** (ответьте и снова проверьте).
4. **Следующая карточка** — переход; прогресс сохраняется в `~/.ai-anki/progress.json`.
